Var

Con la aparicion de la nueva version de javascript "ES2015" aparecieron nuevos elementos: Let y Const. Estos nuevo elementos tienen como fin declarar varibles con mayor facilidad, ya que, antes de la aparicion de estas se utilizaba principalmente "Var", en la cual se encontraron algunos problemas. Var tiene dos ambitos, es decir, donde esta variable esta disponible para su uso, uno de ellos es el ambito global, en el cual "Var" va a funcionar cuando se declare una variable fuera de una funcion, por lo tanto su uso va a estar disponible en toda la pantalla. Otro de los ambitos en donde puede funcionar es el ambito/funcion local, en este caso "Var" funcionara solo cuando se utilize para declarar una variable dentro de una funcion.
